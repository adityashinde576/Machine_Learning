Data Preprocessing Project
Overview

This project demonstrates basic data preprocessing steps using Python, Pandas, Scikit-Learn, and Label Encoding.
The dataset contains employee information such as Name, Department, Experience, and Salary.
The goal is to clean the data, handle missing values, encode categorical features, and scale numerical features.

Features Implemented

Load raw data into a Pandas DataFrame

Identify and fill missing values

Encode categorical variables using LabelEncoder

Standardize numerical columns using StandardScaler

Produce a final processed dataset ready for machine learning

Technologies Used

Python

Pandas

Scikit-Learn

LabelEncoder

StandardScaler
